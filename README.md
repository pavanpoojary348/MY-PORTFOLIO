## 💡 About

My personal portfolio website showcasing my projects, skills, and certifications.
Built with **Flask + Python** and deployed on **Render**.

---

## ✨ Features

- 🏠 Home — Hero section with animated typing effect
- 👨‍💻 About — My story, skills, and journey timeline
- 🚀 Projects — All my projects with Problem → Solution → Result format
- 🧠 Skills — Skill bars with tools & currently learning section
- 🏆 Certificates — Filterable certificate gallery with quick view modal
- 📩 Contact — Working contact form that sends emails directly to me

---
